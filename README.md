# JogoTermo
Modelo de jogo de navegador Termo para aplicação desktop para a disciplina de Programação Orientada a Eventos

Instruções que deveriam ser realizadas: 
  1. Programar back space
  2. Linkar teclado fisico com virtual
  3. Verificar questão da mesma letra em posições difentes
  4. Possibilidade de digitar letra fora de ordem
  5. Painel de progresso
  6. Botão de reiniciar
  7. Verificar se a palavra é valida
  8. Validar palavra menor que 5 letras
  9. Possibilidade e importar palavras online
  10. Efeitos sonoros, click, vitoria, etc.
